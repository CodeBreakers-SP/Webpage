Prototypes

Our prototypes consisted of a login screen and homepage. The login screen is pretty straight forward, consisting of an email and password textfield. The homepage includes several components. On the top-center, a search bar in which the user should be able to search devices and filter results by building, faculty, hardware specfications, and other details. On the right, the client would like to have common acctions that the user would frequently be using. This includes: editing, deleting, duplicating, deactiving, and more. In the center, the user would see relevant information that would change depending on user permissions. On the left side, the user would be the navigation section allowing to navigtation throughout the website.
